      Make this your homepage
      Chrome / Edge
      1. Download extension and unzip it
      2. Go to chrome://extensions
      3. Enable "Developer mode"
      4. Click "Load unpacked" and select the folder
      Firefox
      1. Open Settings → Home
      2. Set "New Windows and Tabs" to "Custom URLs"
      3. Paste this page's URL
      Safari
      1. Open Settings → General
      2. Set "New tabs open with" to "Homepage"
      3. Paste this page's URL in "Homepage"
